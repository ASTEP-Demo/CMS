Introduction

General Data Protection Regulation is generally known as GDPR. The CATO GDPR Search utility enables the administrators to search for text in both CATO and CADMeasure session files. This allows you to search for all the occurrences of an identifying name.  Based on the search results, it is then possible to identify where such information is held so that it can be amended or removed as required. If you require any further assistance, contact the Causeway Support Team.

You can use various comparison methods as described in the Appendix A: Understanding Compare Methods section to search across the CATO database or the CADMeasure session files.

Note: The CATO GDPR Search utility requires a separate licence and you will need to log on using Causeway Licence Manager to access the utility. If you require a licence to access this utility, contact the Causeway Support Team.
